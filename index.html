Import instructions
Reimporting? Google Calendar adds new events on every import — it does not replace existing ones by UID. Always delete the Work Schedule calendar and recreate it before importing again.
1
Create a dedicated calendar for work shifts
On the web (calendar.google.com): click the + next to "Other calendars" in the left sidebar → Create new calendar → name it "Work Schedule" → Save. On mobile: this happens automatically when you sync — no action needed on the phone.
2
Import the .ics file
Go to Settings (gear icon) → Import & Export → Import. Select the downloaded .ics file. In "Add to calendar", choose Work Schedule. Click Import.
3
Check your phone
Open Google Calendar on your phone. Events sync automatically within a few minutes. If they don't appear, pull down to refresh or go to the app's Settings and tap your account to force a sync.
4
When the schedule changes — delete and redo
Go to Settings → Work Schedule → Delete calendar. Create a new one with the same name. Import the new .ics file into it.
Reimporting? Apple Calendar creates duplicates on reimport. Delete the Work Schedule calendar and recreate it before importing a new .ics file.
1
Create a dedicated calendar
Mac: File → New Calendar → name it "Work Schedule". Choose iCloud as the source so it syncs to your iPhone automatically.

iPhone: Open Calendar → tap Calendars at the bottom → Add Calendar → name it "Work Schedule".
2
Import the .ics file
Mac: double-click the downloaded .ics file — Calendar opens and asks which calendar to add events to. Choose "Work Schedule".

iPhone: tap the .ics file in Files or your email/downloads. Calendar opens and asks where to add it. Choose "Work Schedule".
3
Sync across devices
If the calendar is in iCloud, it syncs automatically to all Apple devices signed in to the same Apple ID. Allow a few minutes. To force it: Settings → [your name] → iCloud → Calendars → toggle off and on.
4
When the schedule changes — delete and redo
Mac: right-click "Work Schedule" in the calendar list → Delete. Create it again, import the new file.
iPhone: Calendars → tap ⓘ next to Work Schedule → Delete Calendar. Create a new one and import.
All data stays on your device — nothing is sent to any server.
